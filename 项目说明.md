```
1. 双击 start_gui.bat 启动程序
2. 打开扫雷游戏
3. 点击 "📸 捕获屏幕"
4. 选择棋盘大小
5. 点击 "🔍 分析并提示"
6. 查看绿色（安全）和红色（地雷）提示
7. 根据提示进行游戏
```

## 🔧 依赖包

只需安装4个核心包：

```
opencv-python  # 图像处理
numpy         # 数值计算
pyautogui     # 屏幕截图
Pillow        # 图像绘制
```

安装命令：
```bash
pip install -r requirements.txt
```

## ⚠️ 重要提示

1. **棋盘大小必须正确** - 这是最关键的设置
2. **先翻开几个格子** - 至少3-5个，提供初始信息
3. **窗口清晰可见** - 不要被其他窗口遮挡
4. **5秒内切换** - 点击捕获后要快速切换到游戏窗口
