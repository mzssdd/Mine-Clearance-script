# 快速开始指南

## 第一步：安装依赖

```bash
pip install -r requirements.txt
```

## 第二步：运行演示

先运行演示版本，了解工具的工作原理：

```bash
python minesweeper_demo.py
```

这会：

- ✅ 创建一个模拟的扫雷游戏
- ✅ 自动求解并给出提示
- ✅ 生成两张图片：
  - `demo_initial.png` - 初始状态
  - `demo_hints.png` - 带提示的状态

**查看生成的图片，绿色边框表示安全，红色边框表示地雷！**

## 第三步：使用真实游戏

### 方式一：Windows 扫雷

1. 打开 Windows 扫雷游戏
2. 开始游戏并点击几个格子
3. 运行脚本：

```bash
python minesweeper_solver.py
```

4. 输入 `s` 开始扫描
5. 输入棋盘大小（如 `9x9`）
6. 查看生成的 `minesweeper_hint.png`

### 方式二：在线扫雷

1. 访问 https://minesweeper.online/
2. 选择难度开始游戏
3. 运行上述相同步骤

## 提示符号说明

- 🟢 **绿色边框 + ✓** = 安全格子，建议点击
- 🔴 **红色边框 + 💣** = 地雷格子，建议标记（右键）

## 常见棋盘大小

- **初级 (Beginner)**: `9x9` - 10 个雷
- **中级 (Intermediate)**: `16x16` - 40 个雷
- **高级 (Expert)**: `16x30` - 99 个雷

## 使用技巧

1. **游戏开始时**

   - 先随机点击 3-5 个格子
   - 让 AI 有足够信息分析

2. **扫描时机**

   - 当你不确定下一步时扫描
   - 翻开新格子后重新扫描

3. **提高准确性**
   - 正确输入棋盘大小
   - 确保游戏窗口清晰可见
   - 避免窗口被遮挡

## 问题排查

**识别不准确？**

- 检查棋盘大小输入是否正确
- 确保游戏窗口足够大且清晰
- 尝试调整窗口到屏幕中央

**没有提示？**

- 需要先翻开一些格子
- 当前状态可能需要猜测
- 继续游戏后重新扫描

**找不到游戏窗口？**

- 确保游戏在前台
- 等待扫描时不要移动窗口
- 尝试放大游戏窗口
