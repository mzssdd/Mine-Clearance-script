# 🚀 立即开始使用

## ✅ 问题已修复

所有导入问题已解决，程序现在可以正常运行！

## 📦 第一步：安装依赖

```bash
pip install -r requirements.txt
```

这会安装：
- opencv-python (图像处理)
- numpy (数值计算)
- pyautogui (屏幕截图)
- Pillow (图像绘制)

## 🚀 第二步：启动程序

### 方式一：双击启动（最简单）⭐

**Windows:**
```
双击文件: start_gui.bat
```

**Mac/Linux:**
```bash
chmod +x start_gui.sh
双击文件: start_gui.sh
```

### 方式二：命令行启动

```bash
python run.py
```

### 方式三：VSCode 调试

1. 在 VSCode 中打开项目
2. 按 `F5` 键
3. 选择配置：`Python: 扫雷辅助工具 (GUI)`

## 🎮 第三步：使用程序

### 完整流程：

1. **准备游戏**
   - 打开扫雷游戏（Windows 自带或在线版 https://minesweeper.online/）
   - 随机点击 3-5 个格子开始游戏

2. **捕获屏幕**
   - 在 GUI 窗口中点击 `📸 捕获屏幕 (5秒后)`
   - 立即切换到扫雷游戏窗口（窗口要在最前面）
   - 等待 5 秒自动截图

3. **选择棋盘大小**
   - 在下拉菜单选择：
     - `9x9` - 初级扫雷
     - `16x16` - 中级扫雷
     - `16x30` - 高级扫雷
     - `自定义` - 输入行列数

4. **分析并查看提示**
   - 点击 `🔍 分析并提示` 按钮
   - 查看右侧显示的提示图像：
     - 🟢 **绿色边框 + ✓** = 安全格子，可以点击
     - 🔴 **红色边框 + ✗** = 地雷格子，需要标记（右键）
   - 查看右下角的详细位置信息

5. **根据提示游戏**
   - 点击绿色标记的格子
   - 右键标记红色标记的格子
   - 翻开新格子后，重复步骤 2-4

6. **保存图片（可选）**
   - 点击 `💾 保存图片` 按钮
   - 选择保存位置

## 💡 使用技巧

### ⭐ 最重要：正确设置棋盘大小

**如何确定棋盘大小？**
- 数一下游戏中的行数和列数
- 或根据难度选择：
  - 初级 = 9x9
  - 中级 = 16x16
  - 高级 = 16x30

### 提高准确率

1. **多翻开几个格子**
   - 建议先翻开 5-10 个格子
   - 信息越多，提示越准确

2. **清晰的截图**
   - 游戏窗口要清晰可见
   - 不要被其他窗口遮挡
   - 窗口尽量大一些

3. **及时更新**
   - 每翻开一些新格子后
   - 重新捕获和分析
   - 获取最新提示

## ⚠️ 常见问题

### Q1: 双击 start_gui.bat 没反应？

**解决方案：**
1. 确认已安装 Python 3.7+
2. 运行：`pip install -r requirements.txt`
3. 或直接运行：`python run.py`

### Q2: 截图后是黑屏？

**解决方案：**
- 点击"捕获屏幕"后，**立即**切换到游戏窗口
- 确保游戏窗口在最前面，没有被最小化
- 等待 5 秒让程序自动截图

### Q3: 分析后没有任何提示？

**可能原因和解决方案：**

1. **棋盘大小设置错误** ← 最常见
   - 数一下游戏的实际行列数
   - 重新选择正确的大小

2. **翻开的格子太少**
   - 先多点击几个格子
   - 至少 5-10 个格子

3. **当前状态确实需要猜测**
   - 这很正常
   - 随机点击一个格子后再分析

### Q4: 提示的位置不对？

**解决方案：**
- 检查棋盘大小设置是否正确
- 重新捕获图像
- 确保游戏窗口没有缩放

### Q5: 程序报错？

**检查依赖是否安装：**
```bash
pip list | findstr opencv
pip list | findstr numpy
pip list | findstr pyautogui
pip list | findstr Pillow
```

如果缺少任何一个：
```bash
pip install -r requirements.txt
```
