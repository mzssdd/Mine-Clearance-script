# 🎮 扫雷游戏使用指南

## 🚀 5分钟快速上手

### 第一步：启动游戏

**Windows用户：**
```
双击 start_gui.bat
```

**Mac/Linux用户：**
```bash
chmod +x start_gui.sh
./start_gui.sh
```

### 第二步：选择难度

在窗口顶部选择难度：
- **初级 (9x9)** - 10个地雷，适合新手
- **中级 (16x16)** - 40个地雷，适合进阶
- **高级 (16x30)** - 99个地雷，适合高手

### 第三步：开始游戏

点击 **"🎯 新游戏"** 按钮开始

### 第四步：玩游戏

- **左键点击** 任意格子开始
- **数字** 表示周围8格中有几个地雷
- **右键点击** 标记你认为是地雷的格子
- **继续翻开** 所有安全格子

### 第五步：使用AI提示

当你不确定下一步时：

1. 点击 **"💡 AI提示"** 按钮
2. 查看右侧的提示信息
3. 🟢 **绿色格子** = 安全，可以点击
4. 🔴 **红色格子** = 地雷，应该标记
5. 点击 **"🧹 清除提示"** 清除高亮

## 🎯 游戏目标

翻开所有非地雷格子即可获胜！

## 💡 游戏技巧

### 初学者技巧

1. **从角落开始**
   - 角落格子周围只有3个格子，更安全

2. **理解数字**
   ```
   1 = 周围8格中有1个地雷
   2 = 周围8格中有2个地雷
   ...
   8 = 周围8格全是地雷（极少见）
   ```

3. **标记确定的地雷**
   - 右键标记可以帮助你推理
   - 标记后，剩余地雷数会减少

4. **利用空白区域**
   - 空白格子（没有数字）周围没有地雷
   - 点击空白格会自动翻开周围区域

### 进阶技巧

1. **1-1模式**
   ```
   ? ? ?
   ? 1 1
   ? ? ?
   
   结论：左右两侧各有1个地雷
   ```

2. **1-2-1模式**
   ```
   ? ? ?
   1 2 1
   
   结论：中间3个格子中有2个地雷
   ```

3. **边缘技巧**
   - 边缘格子周围只有5个格子
   - 角落格子周围只有3个格子
   - 更容易推理

### 高手技巧

1. **组合推理**
   - 同时观察多个数字
   - 找出共同的未知格子

2. **排除法**
   - 如果某个地方必须是地雷
   - 那么其他地方就是安全的

3. **概率思维**
   - 当无法100%确定时
   - 选择概率最高的格子

## 🤖 AI提示使用技巧

### 何时使用AI

- ✅ 当你完全不知道下一步时
- ✅ 当你想验证自己的推理时
- ✅ 当你想学习新的推理模式时
- ❌ 不要每一步都使用（会失去乐趣）

### 如何使用AI

1. **先自己推理**
   - 试着找出确定的安全格子
   - 试着找出确定的地雷格子

2. **使用AI验证**
   - 点击"AI提示"按钮
   - 对比AI的结果和你的推理

3. **学习AI的逻辑**
   - 看看AI为什么认为某个格子安全
   - 理解背后的推理规则

4. **清除提示继续**
   - 使用完提示后点击"清除提示"
   - 避免误操作标记的格子

### AI的局限性

- AI只能给出**100%确定**的格子
- 当没有确定的格子时，需要你的运气
- AI不会做概率计算，只做逻辑推理

## 📊 界面说明

### 顶部控制栏

```
[难度选择] [🎯 新游戏] [💡 AI提示] [🧹 清除提示]
```

- **难度选择** - 下拉菜单选择难度
- **新游戏** - 开始新的游戏
- **AI提示** - 获取AI智能提示
- **清除提示** - 清除格子高亮

### 信息显示栏

```
💣 剩余: 10        ⏱️ 时间: 0
```

- **剩余** - 剩余地雷数 = 总地雷数 - 已标记数
- **时间** - 游戏用时（秒）

### 游戏区域

- **左侧** - 游戏棋盘（点击格子）
- **右侧** - AI提示信息（显示安全和地雷位置）

### 格子状态

| 显示 | 含义 |
|------|------|
| 灰色凸起 | 未翻开的格子 |
| 空白 | 已翻开的空白格子 |
| 数字 | 周围地雷数量 |
| 🚩 | 已标记为地雷 |
| 💣 | 地雷（游戏结束时显示） |
| 🟢 绿框 | AI提示：安全格子 |
| 🔴 红框 | AI提示：地雷格子 |

## ⌨️ 快捷操作

- **左键点击** - 翻开格子
- **右键点击** - 标记/取消标记
- **新游戏按钮** - 重新开始

## ❓ 常见问题

### Q: 为什么点击后没有反应？

A: 可能的原因：
- 格子已经被翻开
- 格子已经被标记（右键取消标记后可点击）
- 游戏已经结束

### Q: 为什么AI没有给出提示？

A: 可能的原因：
- 翻开的格子太少，信息不足
- 当前状态需要一定运气，AI无法100%确定
- 建议：多翻开几个格子后再试

### Q: 剩余地雷数显示负数？

A: 不会出现这种情况，标记数不能超过总地雷数

### Q: 如何提高游戏速度？

A: 技巧：
1. 多练习，熟悉常见模式
2. 快速识别确定的格子
3. 合理使用AI提示学习
4. 不要过度思考，相信直觉

### Q: 第一次点击会踩雷吗？

A: 不会！第一次点击永远是安全的，地雷会在第一次点击后生成，并且确保第一次点击的位置及其周围都是安全的。
