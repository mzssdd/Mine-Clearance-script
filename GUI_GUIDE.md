# GUI 版本使用指南

## 🖥️ 界面概览

运行 `python minesweeper_gui.py` 后，会打开一个图形窗口，包含以下区域：

### 顶部控制面板

- **棋盘大小选择** - 选择 9x9、16x16、16x30 或自定义
- **📸 捕获屏幕** - 5 秒后自动截取屏幕
- **🔍 分析并提示** - 分析棋盘并生成提示
- **💾 保存图片** - 保存带提示的图片

### 中间显示区域

- **左侧** - 捕获的原始图像
- **右上** - 带提示标记的图像
  - 🟢 绿色边框 + ✓ = 安全格子
  - 🔴 红色边框 + ✗ = 地雷格子
- **右下** - 详细的提示信息文本

## 📖 详细使用步骤

### 步骤 1：准备游戏

1. **打开扫雷游戏**

   - Windows 自带扫雷
   - 在线扫雷（https://minesweeper.online/）
   - 其他扫雷游戏

2. **开始游戏**
   - 随机点击 3-5 个格子
   - 确保游戏窗口清晰可见

### 步骤 2：启动 GUI 程序

```bash
python minesweeper_gui.py
```

或在 VSCode 中按 `F5`，选择：

- **Python: 扫雷辅助工具 (GUI 界面) ⭐ 推荐**

### 步骤 3：选择棋盘大小

在下拉菜单中选择：

- **9x9** - 初级扫雷
- **16x16** - 中级扫雷
- **16x30** - 高级扫雷
- **自定义** - 输入自定义行列数

### 步骤 4：捕获屏幕

1. 点击 **"📸 捕获屏幕 (5 秒后)"** 按钮
2. 程序开始 5 秒倒计时
3. **立即切换到扫雷游戏窗口**
4. 5 秒后自动截图
5. 左侧会显示捕获的图像

### 步骤 5：分析棋盘

1. 点击 **"🔍 分析并提示"** 按钮
2. 程序自动：
   - 识别棋盘区域
   - 分析每个格子状态
   - 计算安全格子和地雷格子
   - 生成提示图像
3. 查看结果：
   - 右上角显示带标记的图像
   - 右下角显示详细信息

### 步骤 6：使用提示

根据显示的提示进行游戏：

**🟢 绿色边框（安全格子）**

- 可以放心点击
- 查看右下角信息，找到具体位置
- 例如："行 3, 列 5"

**🔴 红色边框（地雷格子）**

- 需要右键标记
- 不要点击！
- 标记后继续游戏

### 步骤 7：保存图片（可选）

1. 点击 **"💾 保存图片"** 按钮
2. 选择保存位置
3. 输入文件名
4. 图片会保存为 PNG 格式

## 💡 使用技巧

### 技巧 1：多次分析

- 每翻开新的格子后，可以重新捕获和分析
- 点击"捕获屏幕" → 等待 5 秒 → 点击"分析并提示"

### 技巧 2：准确的棋盘大小

- 错误的棋盘大小会导致分析不准确
- 可以数一下游戏中实际的行列数
- 或参考游戏难度：
  - 初级 = 9x9
  - 中级 = 16x16
  - 高级 = 16x30

### 技巧 3：清晰的截图

- 确保游戏窗口完整可见
- 不要被其他窗口遮挡
- 避免游戏窗口太小
- 建议全屏或最大化游戏窗口

### 技巧 4：充足的信息

- 开始游戏时，先随机点击 5-10 个格子
- 翻开的格子越多，AI 能给出的提示越准确
- 如果没有提示，继续翻开更多格子后重新分析

### 技巧 5：逻辑推理

- 有时 AI 能给出的提示有限
- 这很正常，说明当前状态需要一些"猜测"
- 可以先按照提示操作，翻开更多格子后再分析

## 🔧 常见问题

### Q1: 窗口打不开？

**可能原因：**

- 缺少依赖库

**解决方案：**

```bash
pip install -r requirements.txt
```

### Q2: 截图全黑或全白？

**可能原因：**

- 5 秒内没有切换到游戏窗口
- 游戏窗口被最小化

**解决方案：**

- 点击"捕获屏幕"后立即切换到游戏窗口
- 确保游戏窗口在最前面

### Q3: 没有找到任何提示？

**可能原因：**

- 棋盘大小设置错误
- 翻开的格子太少
- 当前状态确实需要猜测

**解决方案：**

- 检查并修改棋盘大小
- 翻开更多格子后重新分析
- 随机点击一个格子后再次分析

### Q4: 提示位置不对？

**可能原因：**

- 棋盘大小设置错误
- 图像识别不准确

**解决方案：**

- 确认棋盘大小设置正确
- 重新捕获更清晰的图像
- 调整游戏窗口大小后重试

### Q5: 程序运行很慢？

**可能原因：**

- 截图分辨率太高
- 电脑性能较低

**解决方案：**

- 减小游戏窗口大小
- 关闭其他占用资源的程序

## 🎯 最佳实践

### 推荐工作流程

1. **开局**

   - 打开游戏，随机点击角落的几个格子
   - 启动 GUI 程序

2. **第一次分析**

   - 捕获屏幕
   - 分析并查看提示
   - 按照提示点击安全格子或标记地雷

3. **持续游戏**

   - 每翻开一些新格子后
   - 重新捕获和分析
   - 循环直到游戏结束

4. **保存记录**
   - 如果想保留某个状态
   - 使用"保存图片"功能

### 提高准确率

1. **正确的棋盘大小**

   - 这是最重要的设置
   - 数清楚游戏的行数和列数

2. **充足的初始信息**

   - 至少翻开 5 个格子
   - 最好能翻开 10 个以上

3. **清晰的截图**

   - 游戏窗口尽量大
   - 避免模糊或反光

4. **合适的时机**
   - 当不确定下一步时使用
   - 不要在游戏刚开始就使用
